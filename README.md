## S.U.Y.P (Show Us Your ~~Pugs~~ Picts)
#### /!\ W.I.P, even not an alpha /!\

#### HOW IT WORKS :
See exemple.html for a full exemple, but in short :

```
var suyp;
var picturesUrl = [
	'img/001.jpg',
	'img/002.jpg',
	'img/003.jpg',
	'img/004.jpg'
];

$(function(){
	suyp = new Suyp( $('#head .slider'), {
		'slides': picturesUrl,
		'crop': true,
		'mode': 'fade'
	});	
});
```
